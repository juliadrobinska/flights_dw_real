# Flights Data Warehouse Project

This project demonstrates a simple data warehouse built on real US flight data.

## Data
- Source: US flight delay and cancellation dataset
- Volume: ~3 million records
- Data is NOT included in the repository

## Architecture
- staging schema: raw imported data
- core schema: fact and dimension tables
  - flights (fact)
  - dim_airlines
  - dim_airports

## Technologies
- PostgreSQL
- SQL
- DBeaver
- Git / GitHub

## Goals
- Practice data warehouse modeling
- Practice data loading and validation
- Perform analytical SQL queries
